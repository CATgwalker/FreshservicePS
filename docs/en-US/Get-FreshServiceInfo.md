# Get-FreshServiceInfo

## SYNOPSIS
Returns a Freshservice information.

## SYNTAX

```
Get-FreshServiceInfo [<CommonParameters>]
```

## DESCRIPTION
Returns a Freshservice information via REST API.

## EXAMPLES

### EXAMPLE 1
```
Get-FreshServiceInfo
```

Returns a Freshservice connection information.
